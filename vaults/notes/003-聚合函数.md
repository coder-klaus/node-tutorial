SQL提供聚合函数，直接对数据集合进行计算，比如数量、最大值、最小值、平均值等

```sql
-- 计算平均值
-- 对于字符串类型值，如果是用于字段，可以省略引号
-- 而如果用于子句或值 则必须加引号
-- 推荐一直加上引号
SELECT AVG(price) AS 华为手机平均值 FROM products WHERE brand = '华为';

-- 查询评分最高的
SELECT MAX(score) FROM products;

-- 查询评分最低的
SELECT MIN(score) FROM products;

-- 算所有手机的总投票数
SELECT SUM(vote) FROM products;

-- 算商品总数 => 空值不计入COUNT聚合函数的计算
SELECT COUNT(*) FROM products;

-- 聚合函数先查出集合，再操作（平均、最大、最小、求和、计数）
-- 先查出所有的华为手机，再统计个数
SELECT COUNT(*) FROM products WHERE brand = '华为';
```
