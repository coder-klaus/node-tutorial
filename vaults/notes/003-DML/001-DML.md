## 插入数据

```sql
-- 插入值 => INSERT INTO 表名 (字段列表) VALUES (值列表)
-- 1. 值列表跟字段一一对应
-- 2. 如果值是字符串类型，要加引号，单引号和双引号都可以，推荐使用单引号
-- 3. 日期也要加引号，内部会转换为对应的日期格式后再进行存储
INSERT INTO t_products (title, description, price, publish_time)
VALUES ('iPhone 100', 'iPhone 100, only 998', 998, '2022-09-10');
```



```sql
-- 可以一次性插入多条数据
INSERT INTO t_products (title, description, price, publish_time)
VALUES ('小米99', '小米99, only 888', 888, '2023-10-10'),
       ('华为666', '华为666, only 6666', 6666, '2055-06-06');
```



```sql
-- 如果要插入的数据覆盖了所有字段，并且顺序与表结构一致，可以省略字段名括号
-- 对于自增字段，可以手动进行设置
-- 	+ 如果设置了, 就以手动设置的为主
-- 	+ 如果没有设置，则自动基于之前的字段值自增加一
--    - 所以自增字段，只能是整数类型值，且自增字段通常用作主键或唯一索引
--    - 即使数据记录被删除了，自增值也不会因为删除记录而回退，后续插入会继续递增。
INSERT INTO t_products 
VALUES (7, 'iPhone 200', 'iPhone 100, only 998', 998, '2022-09-10');
```



## 删除数据

```sql
-- 删除 => 直接把表里所有数据清空，非常危险！
DELETE FROM t_products;
```

```sql
-- 删除 => 基于where子句删除特定记录
-- 即使没有对应记录，执行该sql，也是显示成功
DELETE FROM t_products WHERE id = 4;
```



## 更新数据

```sql
-- 更新数据 => UPDATE 表名 SET 字段=新值
-- 没有限制条件，则会将所有记录全部进行更新
UPDATE t_products SET price = 8888;
```

```sql
-- 1. 通过限制条件限制更新的记录
-- 2. 可以同时更新多个字段
-- 3. 对于SQL, 只要没有语法错误，就可以执行通过
--    如果需要更新的记录不存在也不会报错，而是静默失效
UPDATE t_products SET price = 8888, title = '华为至尊版' WHERE id = 8;
```
